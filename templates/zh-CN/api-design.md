## 4. 请求与响应 Schema

每个操作应包括：

- Method、Path/Topic 和版本；
- 身份验证和授权规则；
- 字段名、类型、单位、Null、默认值、校验和语义注释；
- 正常、边界和非法输入示例；
- 响应与错误 Schema；
- 分页、过滤、排序和限制；
- Correlation Key 和 Idempotency Key；
- 隐私分类。
