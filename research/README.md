# Research and related work

This directory keeps the public claims in **Steer the Build** grounded, comparable, and updateable. It is not intended to prove that every individual idea in this field guide is novel. Many of the component ideas—specification before implementation, automated testing, observability, secure delivery, and reusable agent instructions—have strong precedents.

The contribution of this project is the **integrated operating method** developed from cross-domain practice at a scale where implementation volume can no longer be controlled primarily by line-by-line human review.
