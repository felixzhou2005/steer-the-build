# 相关研究与资料

这个目录用于让 **Steer the Build｜掌舵构建** 中的公开结论保持有依据、可比较、可更新。项目并不宣称每一个单独观点都是首次提出。规格先行、自动化测试、可观察性、安全交付和可复用 Agent 指令都有充分的工程先例。

本项目的主要贡献是：把这些能力整合为一套经过跨领域实践形成的**完整控制方法**，尤其面向代码量已经无法主要依靠人工逐行 Review 的 AI 软件工程环境。

## 内容

- [相关工作](related-work.zh-CN.md)：相邻项目、标准、工具和文章。
- [差异化定位](differentiation.zh-CN.md)：本项目主张什么、不主张什么。
- [术语表](glossary.zh-CN.md)：仓库使用的稳定概念。
- [资料审查规则](source-review-policy.zh-CN.md)：外部事实如何选择、引用和更新。
